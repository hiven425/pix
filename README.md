# Pix 📸

**强大的 Windows 截图工具** — 截图 / 长截图 / 贴图 / OCR / 翻译

一款基于 Electron 构建的现代截图工具，模仿 [PixPin](https://pixpin.cn/) 的核心功能与交互体验。

---

## ✨ 功能特性

| 功能 | 说明 |
|:---:|------|
| 📸 **截图** | 区域框选截图，支持拖拽移动和 8 方向调整选区 |
| ✏️ **标注** | 矩形、椭圆、箭头、画笔、文字、马赛克 |
| 📜 **长截图** | 自动滚动 + 智能拼接 |
| 📌 **贴图** | 图片钉在桌面，支持缩放和透明度调节 |
| 🔤 **OCR** | 中英文文字识别（Tesseract.js） |
| 🌐 **翻译** | 内置谷歌翻译 + 自定义 API（DeepL / OpenAI） |

## 🎯 交互亮点

- 🔍 **像素级放大镜** — 8 倍放大 + 像素网格 + 颜色采样
- 🖱️ **选区拖拽移动** — 完成选区后可自由拖动位置
- ↔️ **手柄调整** — 8 个方向手柄精确调整选区大小
- 🎨 **线宽控制** — 3 档线宽切换，适配不同标注场景
- ⌨️ **丰富快捷键** — Enter 复制 / Esc 取消 / 双击复制 / 右键返回
- 💬 **操作反馈** — 复制成功 Toast 提示

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式启动
npm run dev

# 打包
npm run build
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Alt+A` | 截图 |
| `Ctrl+Alt+S` | 长截图 |
| `Enter` / `Ctrl+C` | 复制到剪贴板 |
| `Ctrl+S` | 保存文件 |
| `Ctrl+Z` | 撤销标注 |
| `Esc` | 取消 / 返回上一步 |
| 双击选区 | 快速复制 |
| 右键 | 取消当前操作 |

## 🏗️ 技术栈

- **Electron** — 桌面应用框架
- **Tesseract.js** — OCR 文字识别引擎
- **Canvas API** — 截图渲染与标注
- **Google Translate API** — 翻译服务
